# Olho na Rua — Release Candidate

## Como abrir
1. Extraia o arquivo release-candidate.zip.
2. Abra o arquivo "index.html" em qualquer navegador moderno.
3. Não é necessário servidor, o site funciona localmente.

## Como testar
- Navegue pelas páginas usando mouse ou teclado (TAB / SHIFT+TAB).
- Teste o formulário de Registrar Ocorrência (campos obrigatórios e validação).
- Teste o menu em telas menores (responsivo + acessibilidade).
- Verifique que não há mensagens no console (F12 > Console).
- Verifique que todas as imagens carregam.
